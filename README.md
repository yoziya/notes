## commit format

> `<type>: <subject>`

### type

`notes init`: 初始化布局。

`add`: 新增 note。

`edit`: 修改 note。

`delete` 删除 note。

### subject

subject是commit目的的简短描述，不超过50个字符。

使用中文，表述简练，结尾不加标点。
